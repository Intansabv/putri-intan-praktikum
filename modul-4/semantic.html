Selamat Datang

Ini adalah contoh halaman menggunakan Semantic HTML5 dengan CSS Grid Layout.

Pengertian

Semantic HTML adalah penggunaan elemen HTML yang memiliki makna sesuai isinya, seperti <header>, <nav>, <section>, dan <footer>. Ini membantu browser dan pembaca memahami struktur halaman.

Kesimpulan

Dengan Semantic HTML, kode menjadi lebih rapi, terstruktur, dan mudah dipahami.
